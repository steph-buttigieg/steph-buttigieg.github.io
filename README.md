Static GitHub Pages version of the current Cambridge personal website.

## Files

- `index.html` - page content and structure
- `styles.css` - responsive visual design
- `script.js` - footer year helper
- `assets/` - portrait, logo, CV, and project-report PDFs

## Publish With GitHub Pages

1. Create a repository named `steph-buttigieg.github.io` for a user site, or any repository name for a project site.
2. Push the contents of this folder to that repository.
3. In GitHub, go to `Settings` -> `Pages`.
4. Choose `Deploy from a branch`, then select the default branch and `/root`.
